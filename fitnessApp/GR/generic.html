Κυκλικο προγραμμα γυμναστικης

Πριν ξεκινήσεις το συγκεκριμένο πρόγραμμα, είναι πολύ σημαντικό να έχει πραγματοποιηθεί καλό ζέσταμα με σκοπό της αποφυγή τραυματισμών.

  • Άσκηση 1:
    • Push ups
    • Μια από τις καλύτερες ασκήσεις καθώς δουλεύει στήθος, ώμους, κορμό και τρικέφαλα.
    • 2-4 set
    • 10-15 επαναλήψεις
  • Άσκηση 2:
    • Squats
    • Ο βασιλειάς όλων των ασκήσεων καθώς δουλεύει όλο το κάτω σώμα και το κορμό. Μην ξεχνάμε τα πόδια, αποτελούν τη βάση του σώματος. Δυνατά πόδια δυνατό σώμα.
    • 4-6 set
    • 20-25 επαναλήψεις
  • Άσκηση 3:
    • Pull ups ή australian pull ups
    • Δύο πολύ δυνατές ασκήσεις που θα δουλέψουν τόσο τη πλάτη όσο και τα χέρια.
    • 2-4 set
    • 5-15 επαναλήψεις
  • Άσκηση 4:
    • Push ups(diamond)
    • Βελτιώνει το μεσαίο μέρος του στήθους και τα τρικέφαλα.
    • 2-4 set
    • 5-15 επαναλήψεις
  • Άσκηση 5:
    • Squats(Wide)
    • 4-6 set
    • 20-25 επαναλήψεις
  • Άσκηση 6:
    • Pull ups or australian pull ups(Χέρια κοντά στο στήθος)
    • Σπουδαία άσκηση για τη βελτίωσει των δικεφάλων.
    • 2-4 set
    • 5-10 επαναλήψεις

    Το συγκεκριμένο πρόγραμμα 3-4 φορές την εβδομάδα είναι για όλα τα επίπεδα, μπορεί να σε βοηθήσει στην αρχή για το fitness ταξίδη σου.
